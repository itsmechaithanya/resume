Portfolio

Project 1
SMS Spam detection using machine learning

The project's primary objective is to categorize messages as either spam or ham (non-spam). The implementation involves the utilization of the Naive Bayes algorithm for message classification. This model demonstrates the capability to distinguish between spam and non-spam messages efficiently. Additionally, the model extends its functionality to classify emails, enhancing its utility in a broader context.
Technologies used: Steamlit, Python

Project 2
Integrating cryptography with steganography

This project focuses on the fusion of cryptography and steganography techniques to enhance data security and privacy. By combining these methods, the project aims to secure information through encryption (cryptography) and then conceal it within seemingly innocuous media files (steganography). The integration provides an additional layer of protection, making it more challenging for unauthorized individuals to access sensitive data.
Technologies used: MERN

Project 3
LMS for school and NGO

Develop a comprehensive Learning Management System (LMS) tailored for schools and NGOs.

Features:


Interactive Games: Engage students through educational and interactive games.

Quiz Functionality: Facilitate assessments with a built-in quiz feature.

Attendance Tracking: Streamline attendance management for both students and teachers.

File Uploads (Teacher): Exclusive file upload access for teachers to share materials.

File Downloads (Students): Students can download files uploaded by teachers for seamless learning.

Progress Cards: Monitor and display student progress through easily accessible progress cards.

Attendance Cards: Generate and view attendance records through user-friendly attendance cards.
Technologies used: MERN
